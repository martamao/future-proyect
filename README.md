# 👾 PIA – Virtual Creature Web App

## 📖 Overview

PIA is an interactive web application inspired by classic virtual pet games (like Tamagotchi), designed to demonstrate modern web development skills.

The project focuses on building a dynamic and engaging user experience using **HTML, CSS, and JavaScript**, and will progressively evolve into a full-stack application with backend, database, and security features.


## 🎮 Features (Current)

* 👾 Virtual creature with dynamic stats:

  * Hunger
  * Energy
  * Happiness
  
* 🎯 Interactive actions:
  * Feed
  * Play
  * Sleep
* ⏳ Automatic stat decay over time
* 💾 Persistent state using `localStorage` (progress is saved)
* 📱 Mobile-first responsive layout


## 🚧 Features in Progress (Frontend)

* Visual progress bars for stats
* Creature state feedback (happy, tired, hungry)
* Death and life-state system
* Animations and visual feedback
* Customization (colors and accessories)
* In-game shop interface


## 🔮 Planned Features (Future Development)

* 🔐 User authentication system (login/register)
* ☁️ Cloud persistence with database (MySQL)
* ⚙️ Backend API (Node.js + Express)
* 🛒 Advanced shop system
* 🧪 Security implementation: 
    * Input validation
    * Password hashing
    * Protection against common vulnerabilities (XSS, SQL Injection)
* 📲 Mobile app adaptation (future possibility)


## 🛠️ Technologies Used

### Future Stack

* Node.js
